Formats measurement values within a given text into human friendly format.

Note that the function uses strtod from the C standard library, therefore the parser decimal point char depend on your locale setting.

Parameters
inputThe text to convert as pointer to a const c-string
bufferPointer to a buffer for the formatted result text. Pass NULL to make a 'virtual' write (like snprintf).
bufferLengthLength (number of bytes) of the given buffer
fracsNumber of desired fractional digits [0 .. 9]
magThe magnitude formatting option, one of MANTISSIFY_MAG_t
sipThe SI-Prefix formatting option, one of MANTISSIFY_SIP_t
Returns
The number of written bytes (excluding the c-string zero termination), also when writing virtual.
